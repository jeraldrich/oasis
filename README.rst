Oasis Gov Bad List Scraper
--------
To extract package::
    - (On Linux / OSX) tar xvf oasis.tgz
    - (On Windows) Double click on oasis.tgz
    - the oasis directory will then be created

To install (have python-virtualenv installed):: 
    - sh install.sh
    - sh install-redis.sh

To run::
    - sh run.sh
    - sh run-redis.sh
    - sh run-celery.sh
